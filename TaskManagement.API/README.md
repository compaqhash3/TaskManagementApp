# Task Management Application

## Tech Stack
- Backend: .NET 9 Web API
- Frontend: Angular 20
- Database: SQL Server
- Authentication: Username/Password

## Features
- User authentication
- Task CRUD operations
- Sorting & filtering
- Side-by-side task list and editor

## How to Run
### Backend
1. Update connection string
2. Run database script
3. `dotnet run`

### Frontend
1. `npm install`
2. `ng serve`
